Quick Start

Get started with Ki2Claude in just a few simple steps:

Zero Configuration
Manual Setup

Simply ensure you're logged into Kiro, then start with one command:

docker-compose up -d

The service will start at http://localhost:8989

For manual setup, build and run the Docker container:

# Build the image
docker build -t ki2claude .

# Run the container
docker run -d \
-p 8989:8989 \
-e KIRO_ACCESS_TOKEN=your_token \
-e KIRO_REFRESH_TOKEN=your_refresh_token \
--name ki2claude \
ki2claude

API Examples

List Models

curl -H "Authorization: Bearer ki2api-key-2024" \
http://localhost:8989/v1/models

Chat Example

curl -X POST http://localhost:8989/v1/messages \
-H "Authorization: Bearer ki2api-key-2024" \
-H "Content-Type: application/json" \
-d '{
"model": "claude-sonnet-4-20250514",
"messages": [
{"role": "user", "content": "Hello, introduce yourself"}
],
"max_tokens": 1000
}'
